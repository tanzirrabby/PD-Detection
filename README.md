# EEG-EMG Multimodal Fusion Demo (Parkinson's Detection)

A modern responsive demo app for capstone presentation.

## Tech Stack
- React + Vite
- Tailwind CSS
- Recharts
- Node.js + Express mock backend

## Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start frontend:
   ```bash
   npm run dev
   ```
3. (Optional) Start backend:
   ```bash
   npm run server
   ```

## Pages
- Landing
- Dashboard
- IoT Simulation with live EEG/EMG mock streams
- Reports
- Doctor Consultation
- Disease Information

## Medical Disclaimer
This system is for research and demonstration purposes only. It is not a medical diagnosis tool and does not provide treatment or medication recommendations.
